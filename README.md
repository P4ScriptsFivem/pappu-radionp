# Pappu-RadioNp
QBCore, QBXCore, ESX Radio And inspired by NoPixel 4.0.

## Features
- Available To Set RestrictedChannels With Jobs
- Set MaxFrequency
- Custom RadioItem 
- Config VoiceSystem
